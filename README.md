# Profile Widget

A modern, responsive profile widget for Ioniță Aurel Mihai (@itsiamdev).

## Overview

This is a static profile card displaying personal information, including:
- Profile picture and cover image
- Name and verified badge
- Bio description
- Social media links (GitHub, LinkedIn, X, Instagram)
- Footer with legal links (Privacy Policy, Terms and Conditions)

## Files

- `index.html` - Main profile page
- `privacy-policy.html` - Privacy policy page
- `terms-and-conditions.html` - Terms and conditions page
- `style.css` - Stylesheet for all pages
- `LICENSE` - MIT License
